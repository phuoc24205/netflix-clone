This project is a Netflix clone built with ReactJS and Tailwind CSS, designed to replicate the core features and user interface of Netflix. It fetches data from The Movie Database (TMDb) API to display trending movies, movie details, and allow users to interact with content. The app incorporates features such as lazy loading, scroll loading, and Framer Motion for smooth animations.
